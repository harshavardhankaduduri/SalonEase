# SalonEase

AI-Powered Salon Booking & Queue Management Platform

## Features
- Appointment Booking
- Real-Time Queue Tracking
- JWT Authentication
- Salon & Barber Management
- AI Hairstyle Recommendations
- WebSocket Notifications

## Tech Stack
- React.js
- FastAPI
- PostgreSQL / SQLite
- WebSockets
- JWT Authentication

## Run Locally

### Backend
uvicorn app.main:app --reload

### Frontend
npm install
npm run dev

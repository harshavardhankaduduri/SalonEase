# SalonEase Project Context

SalonEase is a full-stack salon discovery and live queue management platform.

## Core Stack

- Frontend: React, Vite, Tailwind CSS, React Router, Axios, Context API
- Backend: FastAPI, SQLAlchemy, Pydantic, JWT authentication
- Database: PostgreSQL
- Realtime: FastAPI WebSockets
- Deployment: Docker Compose

## MVP Scope

- Customer registration and login
- Salon discovery, search, filtering, and details
- Barber profiles with portfolio images
- Remote queue joining with booking ID and OTP
- Live queue tracking via WebSockets
- Owner salon registration, barber CRUD, OTP verification, and queue management
- Review creation and salon review listing

## Future Features

- Razorpay payments
- AI hairstyle recommendations
- Smart ETA prediction
- Emergency walk-in mode
